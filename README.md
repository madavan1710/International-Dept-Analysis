# 🌍 International Debt Analysis Dashboard

## 📌 Project Overview

The **International Debt Analysis Dashboard** is an interactive data analytics project developed using **Python, Streamlit, PostgreSQL, Pandas, and Plotly**. It provides comprehensive insights into international debt data through interactive dashboards and SQL-based analytical queries.

The project enables users to explore country-wise debt, indicator-wise debt, global debt trends, rankings, cumulative analysis, and contribution analysis using an intuitive web interface.

---

## 🎯 Project Objectives

* Analyze international debt data using SQL.
* Build an interactive dashboard with Streamlit.
* Perform data visualization using Plotly.
* Apply SQL analytical concepts including joins, aggregations, window functions, views, ranking, and filtering.
* Present meaningful business insights from financial data.
